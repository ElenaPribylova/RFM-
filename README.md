RFM‑анализ - метод сегментации клиентской базы по трём ключевым показателям поведения покупателей. 
Его цель - выделить группы клиентов с разным уровнем лояльности и ценности для бизнеса, чтобы выстроить персонализированные маркетинговые стратегии для каждой из них.
•	Recency (R, давность) - как давно клиент покупал
•	Frequency (F, частота) - как часто клиент покупает
•	Monetary (M, денежный вклад) - сколько денег принёс клиент
